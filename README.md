# DDS to PNG Converter

A simple Java utility for converting **DDS image files** to **PNG format**. Built for quick, one-off conversions with minimal user input.

> **Status:** Not actively maintained

---

## Features
- **User-friendly CLI**: Prompts for file path and confirms before processing.
- **Error handling**: Validates input and checks for image corruption.
- **Cross-platform**: Works on any system with Java installed.
